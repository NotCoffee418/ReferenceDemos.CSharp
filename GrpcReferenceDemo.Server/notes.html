gRPC

Checklist for adding new protos

  • Build action must be set to "Protobuf compiler"
  • gRPC Stub classes must define client / server / both acoording to the situation.
  • Make sure new services are defined in Startup.cs
  • Rebuild the project containing the protos.

Good to know

Some voodoo happens behind the scenes to generate the code your services inherit from.
Make sure to rebuild shared if new or updated protos aren't showing up.

gRPC data types are not C# datatypes. See here for reference.
